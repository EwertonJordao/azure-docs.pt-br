# <a name="toggle-live-chat"></a>Alternar chat ao vivo

Este artigo de como fazer mostra como alternar o chat ao vivo em seu aplicativo IoT Central. Você pode usar o chat ao vivo para acessar o suporte técnico.

> [!NOTE]
> A opção de chat está disponível apenas para aplicativos criados usando o plano de preços gratuitos.

## <a name="chat-with-us"></a>Converse conosco

Para obter suporte técnico, abra seu aplicativo IoT Central e selecione **Chat conosco**.

![Converse conosco](media/howto-show-hide-chat/chat-with-us.png)

Você pode digitar uma pergunta como mostrado na captura de tela a seguir:

![Formulário "Configurar detalhes do dispositivo" com detalhes para configurações e propriedades](media/howto-show-hide-chat/sample-chat.png)

## <a name="hide-chat"></a>Ocultar bate-papo

Para ocultar o bate-papo, escolha **Ocultar bate-papo** no painel **Ajuda:**

 ![Ocultar chat](media/howto-show-hide-chat/hide-chat.png)

## <a name="enable-chat"></a>Habilite o bate-papo

Para mostrar o bate-papo, escolha **Mostrar bate-papo** no painel **Ajuda:**

 ![Mostrar bate-papo](media/howto-show-hide-chat/show-chat.png)
