# <a name="how-to-delete-an-invitation-to-a-recipient-in-azure-data-share"></a>Como excluir um convite para um destinatário no Azure Data Share

Este artigo explica como excluir um convite a um destinatário antes que ele o aceite. Isso os impedirá de criar uma assinatura de compartilhamento e eles nunca receberão nenhum dado compartilhado. Se você excluir o convite para um consumidor que já criou uma assinatura de compartilhamento, a exclusão do convite não terá nenhum impacto.

## <a name="navigate-to-a-sent-data-share"></a>Navegue para um compartilhamento de dados enviado

No Azure Data Share, navegue até o seu compartilhamento enviado e selecione a guia **Convites.** Listados aqui estão todos os destinatários de convites para este compartilhamento de dados. Clique nas caixas ao lado dos destinatários cujos convites você gostaria de excluir e, em seguida, clique em **Excluir**.
