# <a name="security-recommendations---a-reference-guide"></a>Recomendações de segurança - um guia de referência

Este artigo lista as recomendações que você pode ver no Azure Security Center. As recomendações mostradas em seu ambiente dependem dos recursos que você está protegendo e de sua configuração personalizada.

Para saber como responder a essas recomendações, consulte [Remediar recomendações no Azure Security Center](security-center-remediate-recommendations.md).

Seu Secure Score é baseado no número de recomendações do Security Center que você completou. Para decidir quais recomendações resolver primeiro, veja a gravidade de cada uma e seu potencial impacto no seu Secure Score.

>[!TIP]
> Se a descrição de uma recomendação diz "Nenhuma política relacionada", geralmente é porque essa recomendação depende de uma recomendação diferente e *de sua* política. Por exemplo, a recomendação "Falhas de saúde de proteção de endpoint devem ser corrigidas...", baseia-se na recomendação de verificar se uma solução de proteção de ponto final está mesmo *instalada* ("Solução de proteção de endpoint deve ser instalada..."). A *recomendação* subjacente tem uma política. Limitar as políticas apenas à recomendação fundamental simplifica a gestão de políticas.
