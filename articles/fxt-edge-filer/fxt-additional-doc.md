# <a name="additional-documentation"></a>Documentação adicional 

Alguns recursos fora deste site docs.microsoft.com podem ajudá-lo a entender e trabalhar com o cache híbrido do Microsoft Azure FXT Edge Filer.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Avere vFXT para documentação de cache do Azure

[Avere vFXT for Azure](https://docs.microsoft.com/azure/avere-vfxt/) é um produto de cache de cluster semelhante que usa nóvirtual no Microsoft Azure. Ele é criado com o portal Azure, mas usa o mesmo software de configuração do painel de controle que o cache de hardware Azure FXT Edge Filer. 

## <a name="legacy-hardware-documentation"></a>Documentação de hardware legado 

Esta tecnologia foi usada anteriormente em produtos da Avere Systems. A documentação do usuário para esses produtos mais antigos está disponível no [site da Avere Legacy Documentation](https://azure.github.io/Avere/). 

Especificamente, esses documentos podem ter detalhes úteis:

* [Guia de configuração de cluster](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) - Inclui informações de visão geral sobre tarefas comuns de configuração e gerenciamento de clusters e tem explicações detalhadas das páginas de configurações de cluster 

* [Guia do Painel](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) - Explica as características da guia **Painel do** painel de controle

* [FXT Cluster Creation Guide](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) - Guia de criação de clusters de produtos anteriores
