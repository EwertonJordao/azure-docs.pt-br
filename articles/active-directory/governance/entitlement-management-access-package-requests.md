# <a name="view-requests-for-an-access-package-in-azure-ad-entitlement-management"></a>Exibir solicitações para um pacote de acesso no gerenciamento de direitos ad do Azure

Na gestão de direitos do Azure AD, você pode ver quem solicitou pacotes de acesso, sua política e status. Este artigo descreve como visualizar solicitações de pacotes de acesso.

## <a name="view-requests"></a>Exibir solicitações

**Papel pré-requisito:** Administrador global, administrador de usuário, proprietário de catálogo ou gerenciador de pacotes de acesso

1. No portal do Azure, clique em **Azure Active Directory** e, em seguida, em **Governança de Identidade**.

1. No menu à esquerda, clique em **Acessar pacotes** e, em seguida, abra o pacote de acesso.

1. Clique **em Solicitações**.

1. Clique em uma solicitação específica para ver detalhes adicionais.

    ![Lista de pedidos para um pacote de acesso](./media/entitlement-management-access-package-requests/requests-list.png)
