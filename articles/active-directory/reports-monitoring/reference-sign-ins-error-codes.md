# <a name="sign-in-activity-report-error-codes"></a>Códigos de erro do relatório de atividade de login 

Com as informações fornecidas pelo [relatório de login do usuário,](concept-sign-ins.md)você encontra respostas para perguntas como:

- Quem entrou na minha inscrição?
- Quais aplicativos foram conectados?
- Quais logins falharam e por quê?

Quando um login falha, você vê um código de erro correspondente à falha. [Encontre a descrição de um código de erro](https://login.microsoftonline.com/error).
