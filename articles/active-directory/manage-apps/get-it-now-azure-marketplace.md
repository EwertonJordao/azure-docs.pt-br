# <a name="get-it-now---add-an-app-from-the-azure-marketplace"></a>Obter Agora – adicionar um aplicativo do Azure Marketplace

Você está quase lá! 

Caso esteja tentando usar o Azure AD como o provedor de identidade de um aplicativo, então você está no lugar certo. Você só precisa adicioná-lo ao seu locatário do Azure AD. Para saber como fazer isso, siga a série de guias de início rápido aqui: [Exibir aplicativos em seu locatário do Azure AD](view-applications-portal.md).

Caso esteja tentando instalar um aplicativo em seu computador local ou dispositivo móvel, então você está no lugar errado. O Azure Marketplace foi projetado para organizações que usam aplicativos com seus locatários do Azure. Para computadores e dispositivos pessoais, confira [os aplicativos da Microsoft Store](https://www.microsoft.com/store/apps).
