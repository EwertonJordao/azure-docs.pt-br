# <a name="troubleshoot-azure-migrate"></a>Solucionar problemas das Migrações para Azure

[O Azure Migrate](migrate-services-overview.md) fornece um hub de ferramentas para avaliação e migração, bem como ofertas de fornecedores independentes de software (ISV) de terceiros. Este artigo ajuda você a solucionar problemas com o Azure Migrate, a Avaliação do Servidor Migração do Azure e a migração do servidor Azure Migrate.

## <a name="how-do-i-create-or-find-a-project"></a>Como crio ou encontro um projeto?

Revise o [guia de solução de problemas do projeto Azure Migrate](troubleshoot-project.md).

## <a name="i-cant-get-the-appliance-working"></a>Não consigo fazer o aparelho funcionar.

Revisar [respostas a problemas comuns](troubleshoot-appliance-discovery.md) com a implantação do aparelho.

## <a name="machines-arent-discovered"></a>Máquinas não são descobertas

Revise problemas comuns de descoberta.

## <a name="app-discovery-isnt-working"></a>A descoberta de aplicativos não está funcionando

A descoberta de aplicativos, funções e recursos em execução em máquinas locais atualmente só é suportada para VMs VMware. [Revise erros comuns](troubleshoot-appliance-discovery.md#common-app-discovery-errors) para detecção de aplicativos.

## <a name="assessment-isnt-working"></a>A avaliação não está funcionando.

[Revise](troubleshoot-assessment.md) problemas e erros comuns de avaliação.
