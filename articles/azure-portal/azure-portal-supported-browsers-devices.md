# <a name="supported-devices"></a>Dispositivos com suporte

O [portal Azure](https://portal.azure.com) é um console baseado na Web e funciona no navegador de todos os desktops e dispositivos tablet modernos. Se você precisa gerenciar os recursos do Azure a partir de um dispositivo móvel, experimente o [aplicativo móvel Azure](https://azure.microsoft.com/features/azure-portal/mobile-app/). Está disponível para iOS e Android.

## <a name="recommended-browsers"></a>Navegadores recomendados

É recomendável que você use o navegador mais atualizado compatível com o sistema operacional. Há suporte para os seguintes navegadores:

* Microsoft Edge (última versão)
* Internet Explorer 11
* Safari (última versão, apenas Mac)
* Chrome (última versão)
* Firefox (última versão)
