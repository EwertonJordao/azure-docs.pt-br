# <a name="azure-spatial-anchors-support-options"></a>Opções de suporte a âncoras espaciais do Azure

## <a name="open-a-tech-support-ticket"></a>Abra um ticket de suporte técnico

Para abrir um bilhete de suporte técnico dentro do Portal Azure para âncoras espaciais azure:

1. Com o [portal Azure](https://azure.microsoft.com/account/) aberto, selecione o ícone de ajuda na barra de menu superior e selecione o botão **Ajuda + suporte.**

![Ajuda e suporte do portal Azure](./media/spatial-anchor-support.png)

1. Com a página ajuda + suporte aberta, selecione **+ Nova solicitação de suporte**.

![Novo pedido de suporte do portal Azure](./media/spatial-anchor-support2.png)

1. Ao completar os campos de bilhetes de suporte:

- Tipo de problema: técnico
- Serviço: Âncoras Espaciais

![Campos de bilhetes de suporte do portal Azure](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>Suporte da comunidade

Para obter suporte da comunidade de usuários, consulte [âncoras espaciais do Azure no Stack Overflow](https://stackoverflow.com/questions/tagged/azure-spatial-anchors).

## <a name="provide-content-feedback"></a>Forneça feedback de conteúdo

Na parte inferior de cada artigo de conteúdo, há a oportunidade de abrir um problema do GitHub e fornecer feedback sobre o conteúdo da documentação do Azure Spatial Anchor.

## <a name="provide-product-feedback"></a>Fornecer comentários sobre o produto

Para fornecer feedback, compartilhar uma ideia ou sugestão para o serviço de Âncoras Espaciais do Azure, ou votar nas ideias que outros apresentaram, visite o [Fórum de Feedback de Âncoras Espaciais do Azure.](https://feedback.azure.com/forums/919252-azure-spatial-anchors)
