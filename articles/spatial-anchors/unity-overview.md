# <a name="building-in-unity-with-azure-spatial-anchors"></a>Construção em Unidade com Âncoras Espaciais Azure

Os desenvolvedores podem escolher o Unity para criar e implantar aplicativos de realidade mista que usam âncoras espaciais do Azure. As seguintes plataformas são suportadas:

**Unidade para HoloLens**

[Quickstart: Crie um aplicativo Unity HoloLens que usa âncoras espaciais do Azure](https://docs.microsoft.com/azure/spatial-anchors/quickstarts/get-started-unity-hololens)

**Unidade para Android**

[Quickstart: Crie um aplicativo unity Android que usa âncoras espaciais do Azure](https://docs.microsoft.com/azure/spatial-anchors/quickstarts/get-started-unity-android)

**Unidade para iOS**

[Quickstart: crie um aplicativo unity iOS que usa âncoras espaciais do Azure](https://docs.microsoft.com/azure/spatial-anchors/quickstarts/get-started-unity-ios)
