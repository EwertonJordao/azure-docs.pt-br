# <a name="index-of-built-in-security-controls-articles-for-azure-services"></a>Índice de artigos "Controles de segurança incorporados" para serviços do Azure

Este índice fornece links para artigos "Controles de segurança incorporados". Um controle de segurança é uma qualidade ou característica de um serviço Azure que contribui para a capacidade do serviço de prevenir, detectar e responder a vulnerabilidades de segurança.

- [Tecido de serviço azure](../../service-fabric/service-fabric-security-controls.md)
- [Banco de dados SQL do Azure](../../sql-database/sql-database-security-controls.md)
- [Conjuntos de dimensionamento de máquina virtual do Azure](../../virtual-machine-scale-sets/virtual-machine-scale-sets-security-controls.md)
