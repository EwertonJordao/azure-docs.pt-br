# <a name="monitor-and-manage-wunderlist-by-using-azure-logic-apps"></a>Monitore e gerencie o Wunderlist usando os Aplicativos Lógicos do Azure

A partir de 6 de maio de 2020, o [serviço do Wunderlist foi desligado](https://www.wunderlist.com/) e solicitou que os clientes [migrassem seus dados e comecem a usar a Microsoft para fazer isso](https://www.wunderlist.com/blog/goodbye-from-wunderlist/). Você pode usar o [conector da Microsoft para fazer](/connectors/todo/) para substituir o [conector do Wunderlist](/connectors/wunderlist/)preterido.
