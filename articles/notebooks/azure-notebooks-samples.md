# <a name="discover-and-use-sample-notebooks-in-azure-notebooks-preview"></a>Descobrir e usar notebooks de exemplo na versão prévia do Azure Notebooks

A Internet tem muitas coleções do Jupyter Notebooks interessantes e úteis que você pode trazer facilmente para o Azure Notebooks. Este artigo fornece uma lista de exemplos notáveis.

[!INCLUDE [notebooks-status](../../includes/notebooks-status.md)]

Após identificar um notebook que você gostaria de explorar, aqui estão alguns detalhes que você vai querer conhecer:

- Se um notebook já estiver no Azure Notebooks, use o botão **Clonar** no painel do projeto para fazer uma cópia do projeto na sua própria conta.
- Alguns notebooks exigem pacotes específicos para serem executados. Embora seja possível usar `!pip install` (Python) ou `install.packages` (R) para instalar pacotes de um notebook, você também pode [configurar um projeto do Azure Notebooks com o ambiente apropriado](configure-manage-azure-notebooks-projects.md).

## <a name="great-sample-notebooks"></a>Excelentes notebooks de exemplo

- O [conjunto inicial do Azure Notebooks](https://notebooks.azure.com/#sample-redirect) inclui introduções ao Python, ao R e ao F#, junto com notebooks que demonstram o acesso a dados, o Azure Machine Learning e alguns exercícios de ciência de dados.
- [Introdução ao Jupyter Notebooks](https://nbviewer.jupyter.org/github/jupyter/notebook/blob/master/docs/source/examples/Notebook/Notebook%20Basics.ipynb) (jupyter.org)
- [Introdução à Ciência de Dados](https://github.com/jakevdp/PythonDataScienceHandbook/tree/master/notebooks), por Jake Vanderplas.
- [Galeria de notebooks interessantes](https://github.com/ipython/ipython/wiki/A-gallery-of-interesting-IPython-Notebooks) (Projeto do Jupyter no GitHub)
- [nbviewer.jupyter.org](https://nbviewer.jupyter.org)
- Instruções de física do [Departamento de Física do Imperial College of London](https://notebooks.azure.com/cvanbreu/libraries)
- [Exemplos de introdução ao Azure Machine Learning](https://notebooks.azure.com/azureml/projects/azureml-getting-started)
