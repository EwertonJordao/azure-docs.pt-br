# <a name="set-up-support-information-lab-account-owner-in-azure-lab-services"></a>Configure informações de suporte (dono de conta de laboratório no Azure Lab Services)
Este artigo explica como você (como proprietário de uma conta de laboratório) pode especificar informações de suporte que criadores de laboratório (instrutores) e usuários de laboratório (estudantes) podem usar para obter ajuda se eles encontrarem quaisquer problemas técnicos com o uso da conta de laboratório ou laboratório. 

As informações de suporte incluem:

- URL
- Email
- Telefone
- Instruções adicionais 

## <a name="specify-support-information"></a>Especificar informações de suporte
1. Faça login no [portal Azure](https://portal.azure.com).
2. Na barra de pesquisa, insira **Serviços de Laboratório**e selecione **Serviços de Laboratório** nos resultados de pesquisa. 
3. Selecione sua conta de laboratório na lista de contas de laboratório. 
4. Mude para a página **de suporte interno,** faça as seguintes etapas:
    1. Digite a **URL de suporte**. 
     2. Digite o **e-mail de suporte**. 
     3. Digite o **telefone de suporte**.
     4. Digite **instruções detalhadas de suporte** (opcional). Os proprietários e usuários do laboratório verão este texto juntamente com as informações de contato de suporte. Os URLs serão automaticamente transformados em links. 
     5. Selecione **Salvar** na barra de ferramentas.
