# <a name="view-support-information-lab-user-in-azure-lab-services"></a>Exibir informações de suporte (usuário de laboratório no Azure Lab Services)
Este artigo explica como você (como um usuário do laboratório) pode exibir as seguintes informações de suporte:

- URL
- Email
- Telefone
- Instruções adicionais

Você pode usar essas informações para obter ajuda quando encontrar problemas técnicos ao usar um laboratório em uma conta de laboratório.

 
## <a name="view-support-information"></a>Exibir informações de suporte
1. Entre no [portal do Azure Lab Services](https://labs.azure.com).
2. Selecione o **laboratório ou a máquina virtual** para a qual você precisa de ajuda e selecione **?** no canto superior direito da página. 
3. Confirme que você vê links para o **site de suporte de exibição**, **suporte por email**e **número de telefone de suporte**.

    ![Exibir informações de suporte](../media/lab-user-support-information/support-information.png)
4. Você pode exibir informações de contato de suporte para outro laboratório alternando para esse laboratório na lista suspensa. 

    ![Alternar para outro laboratório](../media/lab-user-support-information/switch-another-lab.png)
5. Agora, você vê as informações de contato de suporte para o outro laboratório. 

    ![Informações de suporte do outro laboratório](../media/lab-user-support-information/second-lab-support-information.png)
