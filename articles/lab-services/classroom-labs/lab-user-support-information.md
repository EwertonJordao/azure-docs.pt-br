# <a name="view-support-information-lab-user-in-azure-lab-services"></a>Ver informações de suporte (usuário de laboratório no Azure Lab Services)
Este artigo explica como você (como usuário de laboratório) pode visualizar as seguintes informações de suporte:

- URL
- Email
- Telefone
- Instruções adicionais

Você pode usar essas informações para obter ajuda quando você se deparar com quaisquer problemas técnicos ao usar um laboratório em uma conta de laboratório.

 
## <a name="view-support-information"></a>Exibir informações de suporte
1. Faça login no [portal azure Lab Services](https://labs.azure.com).
2. Selecione o **laboratório ou a máquina virtual** para a qual você precisa de ajuda e selecione **?** no canto superior direito da página. 
3. Confirme se você vê links para o site de suporte à **visualização,** **suporte por e-mail**e **número de telefone de suporte.**

    ![Exibir informações de suporte](../media/lab-user-support-information/support-information.png)
4. Você pode visualizar informações de contato de suporte para outro laboratório, mudando para aquele laboratório na lista de baixa. 

    ![Mude para outro laboratório](../media/lab-user-support-information/switch-another-lab.png)
5. Agora, você vê as informações de contato de suporte para o outro laboratório. 

    ![Informações de suporte de outros laboratórios](../media/lab-user-support-information/second-lab-support-information.png)
