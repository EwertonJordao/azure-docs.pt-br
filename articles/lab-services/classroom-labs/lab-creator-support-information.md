# <a name="view-support-information-lab-creator-in-azure-lab-services"></a>Ver informações de suporte (criador de laboratório no Azure Lab Services)
Este artigo explica como você (como criador de laboratório) pode visualizar as seguintes informações de suporte:

- URL
- Email
- Telefone
- Instruções adicionais

Você pode usar essas informações para obter ajuda quando encontrar quaisquer problemas técnicos enquanto cria um laboratório em uma conta de laboratório.

 
## <a name="view-support-information"></a>Exibir informações de suporte
1. Faça login no [portal azure Lab Services](https://labs.azure.com).
2. Selecione ponto de interrogação **(?**) no canto superior direito da página. 
3. Confirme se você vê links para o site de suporte à **visualização,** **suporte por e-mail**e **número de telefone de suporte.**

    ![Exibir informações de suporte](../media/lab-creator-support-information/support-information.png)
