# <a name="azure-data-factory-whitepapers"></a>Whitepapers da Fábrica de Dados Azure 

[!INCLUDE[appliesto-adf-xxx-md](includes/appliesto-adf-xxx-md.md)]

Os whitepapers permitem explorar a Fábrica de Dados Azure em um nível mais profundo. Este artigo fornece uma lista de whitepapers disponíveis para a Fábrica de Dados Azure.

| **Whitepaper** | **Descrição** |
| --- | --- |
|[Fábrica de dados do Azure — Integração de dados na nuvem](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | Este artigo descreve como a Fábrica de Dados do Azure pode permitir que você construa um moderno data warehouse, habilite análises avançadas para conduzir aplicativos SaaS inteligentes e elevar seus pacotes de serviços de integrações de servidores SQL para o Azure.|
|[Migração de dados do Data Warehouse relacional no local para o Azure usando a Fábrica de Dados do Azure](https://azure.microsoft.com/resources/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/) | Este artigo aborda a complexidade da migração de dezenas de dados de TB do data warehouse relacional existente (por exemplo, Netezza, Oracle, Teradata, servidor SQL) para o Azure (por exemplo, Blob Storage ou Azure Data Lake Storage) usando o Azure Data Factory. Os desafios e as melhores práticas são ilustrados em torno da resiliência, desempenho, escalabilidade, gerenciamento e segurança para a jornada de ingestão de big data para o Azure pela Azure Data Factory. |
|[Integração de dados SAP usando a Fábrica de Dados Do Azure](https://github.com/Azure/Azure-DataFactory/blob/master/whitepaper/SAP%20Data%20Integration%20using%20Azure%20Data%20Factory.pdf) | Este artigo resume o suporte atual da Azure Data Factory à integração de dados SAP, incluindo cenário-alvo, opções de conector SAP e comparação para diferentes requisitos e introdução em cada conector SAP na Fábrica de Dados. | 
|[Fábrica de dados do Azure: SSIS na nuvem](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| Este artigo explica por que você gostaria de migrar suas cargas de trabalho existentes do SSIS para a Fábrica de Dados Do Azure e abordar considerações e preocupações comuns. Em seguida, mostraremos os detalhes técnicos da criação de um IR Azure-SSIS e, em seguida, mostraremos como carregar, executar e monitorar seus pacotes através da Fábrica de Dados Do Azure usando as ferramentas que você provavelmente está familiarizado como o SQL Server Management Studio (SSMS). |
|[Fábrica de dados Azure - Parâmetros de passagem](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| Este artigo orienta como passar parâmetros entre um pipeline e atividade, bem como entre as atividades. |
|[Fábrica de dados Azure – DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| Este whitepaper abrange algumas das melhores práticas em torno da integração contínua e implantação da Fábrica de Dados Azure. |
