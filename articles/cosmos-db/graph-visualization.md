# <a name="visualize-graph-data-stored-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Visualize dados gráficos armazenados na API AZURE Cosmos DB Gremlin com soluções de visualização de dados

Você pode visualizar dados armazenados na API Azure Cosmos DB Gremlin usando várias soluções de visualização de dados. As seguintes soluções são recomendadas pela [comunidade Apache Tinkerpop](https://tinkerpop.apache.org/#poweredby) para visualização de dados gráficos.

## <a name="linkurious-enterprise"></a>Empresa Linkurious

![Logotipo Linkurious](./media/graph-visualization/linkurious-logo.jpg)

[A Linkurious Enterprise](https://linkurio.us/product/) usa tecnologia gráfica e visualização de dados para transformar conjuntos de dados complexos em redes visuais interativas. A plataforma se conecta às suas fontes de dados e permite que os investigadores naveguem perfeitamente através de bilhões de entidades e relacionamentos. O resultado é uma nova capacidade de detectar relacionamentos suspeitos sem fazer malabarismos com consultas ou tabelas.

A interface interativa da Linkurious Enterprise oferece uma maneira fácil de investigar dados complexos. Você pode procurar entidades específicas, expandir conexões para descobrir relacionamentos ocultos e aplicar layouts de sua escolha para desembaraçar redes complexas. A Linkurious Enterprise agora é compatível com a API Azure Cosmos DB Gremlin. É adequado para cenários de visualização de gráficos de ponta a ponta e suporta recursos de leitura e gravação a partir da interface do usuário. Você pode solicitar uma [demonstração de Linkurious com o Azure Cosmos DB](https://linkurio.us/contact/)

![Demonstração Linkurious](./media/graph-visualization/linkurious-demo.gif)

## <a name="cambridge-intelligence"></a>Inteligência de Cambridge

![Logotipo da Cambridge Intelligence](./media/graph-visualization/ci-logo.png)

[Os](https://cambridge-intelligence.com/products/) kits de ferramentas de visualização gráfica da Cambridge Intelligence agora suportam o Azure Cosmos DB. Os dois kits de ferramentas de visualização a seguir são suportados pelo Azure Cosmos DB:

- [KeyLines para desenvolvedores JavaScript](https://cambridge-intelligence.com/keylines/)

- [Re-Graph para desenvolvedores React](https://cambridge-intelligence.com/regraph/)

![Demonstração de Inteligência de Cambridge](./media/graph-visualization/ci-demo-2.gif)

Esses kits de ferramentas permitem que você projete aplicações de visualização e análise de gráficos de alto desempenho para o seu caso de uso. Eles aproveitam a poderosa renderização da Web Graphics Library (WebGL) e código cuidadosamente criado para dar aos usuários uma experiência de visualização rápida e perspicaz. Essas ferramentas são compatíveis com qualquer navegador, dispositivo, servidor ou banco de dados, e vêm com tutoriais passo a passo, APIs totalmente documentadas e demonstrações interativas.

![Demonstração de Inteligência de Cambridge](./media/graph-visualization/ci-demo-1.gif)
