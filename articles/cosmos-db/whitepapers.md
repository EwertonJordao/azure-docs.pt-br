| **Whitepaper** | **Descrição** |
| --- | --- |
|[Indexação independente de esquema com Azure Cosmos DB](https://www.vldb.org/pvldb/vol8/p1668-shukla.pdf) | Este documento descreve o subsistema de indexação do Azure Cosmos DB. | [Azure Cosmos DB e dados pessoais](https://servicetrust.microsoft.com/ViewPage/TrustDocuments?command=Download&downloadType=Document&downloadId=87cc6456-4b23-473c-94d3-6c713b8b8956&docTab=6d000410-c9e9-11e7-9a91-892aae8839ad_FAQ_and_White_Papers)| Este documento fornece orientação para os clientes do Azure Cosmos DB gerenciando um banco de dados baseado em nuvem, um banco de dados local, ou ambos, e que precisam garantir que os dados pessoais em seus sistemas de banco de dados sejam tratados e protegidos de acordo com as regras atuais. |
