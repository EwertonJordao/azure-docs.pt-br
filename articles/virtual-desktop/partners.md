# <a name="windows-virtual-desktop-partner-integrations"></a>Integrações de parceiros de desktop virtual do Windows

Este artigo lista provedores de parceiros aprovados e fornecedores de software independentes para o Windows Virtual Desktop.

## <a name="citrix"></a>Citrix

![Logotipo citrix](./media/partners/citrix.png)

O Citrix é um provedor aprovado que oferece às empresas gerenciamento híbrido centralizado de cargas de trabalho de aplicativos virtuais e desktops no Azure, lado a lado com implantações no local. O Citrix Workspace com o serviço Virtual Apps and Desktops permite que os usuários acessem aplicativos e desktops de qualquer dispositivo, aproveitando o avançado protocolo Citrix HDX para oferecer uma experiência de alta definição de qualquer lugar.

O Citrix amplia o valor do Windows Virtual Desktop com ferramentas corporativas robustas para melhorar a densidade e o desempenho do usuário, provisionar cargas de trabalho demanda e simplificar o gerenciamento de imagens e aplicativos. A TI pode otimizar custos com ferramentas inteligentes de dimensionamento, ao mesmo tempo em que oferece uma incrível experiência de usuário testada em campo contra os aplicativos mais difíceis entre os setores. Além disso, o Citrix Managed Desktops é um programa de desktops virtual habilitados para desktops como serviço do Windows que fornece uma solução de gerenciamento simples e baseada em nuvem para fornecer aplicativos e desktops virtuais para qualquer dispositivo.

- [Acesse o site do parceiro](https://more.citrix.com/wvd).

## <a name="vmware"></a>VMware

![Logotipo da VMware](./media/partners/vmware.png)

O VMware Horizon Cloud no Microsoft Azure oferece às organizações a capacidade de conectar sua própria instância do Microsoft Azure ao plano de controle horizon cloud simples e intuitivo, criando uma solução segura, abrangente e hospedada na nuvem para fornecer virtualizada Aplicativos do Windows e desktops. Com o lançamento do Windows Virtual Desktop, a VMware fez uma parceria com a Microsoft para estender a funcionalidade do Windows Virtual Desktop aos clientes que usam o Horizon Cloud no Microsoft Azure. Essa funcionalidade estará disponível no Tech Preview até o final de 2019.
 
Como um provedor aprovado, os benefícios do Windows Virtual Desktop serão estendidos ao VMware Horizon Cloud nos clientes do Microsoft Azure, incluindo o Windows 10 Enterprise para várias sessões, o Windows 7 com atualizações de segurança estendidas gratuitas por até três anos e O FSlogix. Além disso, os clientes receberão os recursos de classe corporativa do Horizon Cloud, como suporte amplo de ponto final; mediação global otimizada para nuvem; opções e configurações de desktop flexíveis; gerenciamento de ambiente de usuário; e suporte para ambientes híbridos.

- [Registre-se para a visualização tecnológica do Horizon Cloud no Microsoft Azure](https://www.vmware.com/learn/323526_TECH_PREVIEW_REG.html).
- [Acesse o site do parceiro](https://www.vmware.com/products/horizon-cloud-virtual-desktops.html).
