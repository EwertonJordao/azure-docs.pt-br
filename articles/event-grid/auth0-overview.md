# <a name="auth0-partner-topics"></a>Tópicos de parceiro do Auth0
![Logotipo do Auth0](./media/auth0-overview/auth0-logo.png)

Auth0, a plataforma de identidade para construtores de aplicativos, fornece aos desenvolvedores e empresas os blocos de construção necessários para proteger seus aplicativos.

O tópico de parceiro do Auth0 permite que você use eventos emitidos pelo sistema do Auth0 para realizar várias tarefas. Entre em contato com os usuários de maneiras significativas após a autenticação ou automatize as tarefas de segurança e infraestrutura.

A integração permite transmitir os eventos de log do Auth0 com alta confiabilidade no Azure. Nela você pode consumir os eventos com seus recursos favoritos do Azure. Essa integração permite reagir a eventos, obter insights, monitorar problemas de segurança e interagir com outros pipelines de dados poderosos.

Para organizações que usam o Auth0 e o Azure, essa integração permitirá que você integre perfeitamente dados em toda a pilha. 
 
## <a name="available-event-types"></a>Tipos de evento disponíveis
A lista completa de tipos de eventos do Auth0 disponíveis e suas descrições estão disponíveis [aqui](https://auth0.com/docs/logs/references/log-event-type-codes).

## <a name="use-cases"></a>Casos de uso

### <a name="engage-with-your-users"></a>Envolva-se com seus usuários
Oferecer uma experiência de usuário sólida é crítico para reduzir a rotatividade e manter seus usuários. Forneça mais experiências de aplicativos personalizadas usando eventos do Auth0 com Azure Functions e Aplicativos Lógicos. 

### <a name="understand-user-behavior"></a>Noções básicas do comportamento do usuário
Entenda quando os usuários acessam seu produto, onde eles estão conectados e quais dispositivos eles usam. Desenvolva um reconhecimento das áreas de produtos mais importantes rastreando esses sinais. Esses sinais ajudam a determinar quais navegadores e dispositivos oferecem suporte a idiomas para localizar seu aplicativo e quando são seus horários de pico de tráfego. 

### <a name="manage-user-data"></a>Gerenciar dados do usuário
Manter e auditar as ações do usuário é crucial para manter a segurança e as seguintes regulamentações do setor. A capacidade de editar, remover ou exportar dados do usuário está se tornando cada vez mais importante para seguir leis de privacidade, como o Regulamento Geral sobre a Proteção de Dados (GDPR) da União Europeia.

### <a name="secure-your-application"></a>Proteger seu aplicativo
É importante a combinação de procedimentos de monitoramento de segurança e resposta a incidentes ao proteger um sistema distribuído. Por isso, também é importante manter todos os dados em um único lugar e monitorar toda a pilha.
