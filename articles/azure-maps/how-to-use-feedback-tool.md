# <a name="provide-data-feedback-to-azure-maps"></a>Forneça feedback de dados ao Azure Maps

O Azure Maps está disponível desde maio de 2018. O Azure Maps tem fornecido novos dados de mapa, APIs rest fáceis de usar e SDKs poderosos para oferecer suporte aos nossos clientes corporativos com diferentes tipos de casos de uso de negócios. O mundo real está mudando a cada segundo, e é crucial para nós fornecer uma representação digital factual para nossos clientes. Nossos clientes que estão planejando abrir ou fechar instalações precisam de nossos mapas para atualizar prontamente. Assim, eles podem planejar eficientemente a entrega, manutenção ou atendimento ao cliente nas instalações certas. Criamos o site de feedback de dados do Azure Maps para capacitar nossos clientes a fornecer feedback direto de dados. O feedback dos dados dos clientes vai diretamente para nossos provedores de dados e seus editores de mapas. Eles podem avaliar e incorporar feedback rapidamente em nossos produtos de mapeamento.  

[O site de feedback de dados do Azure Maps](https://feedback.azuremaps.com) fornece uma maneira fácil para nossos clientes fornecerem feedback de dados de mapa, especialmente em pontos de negócios de interesse e endereços residenciais. Este artigo orienta você sobre como fornecer diferentes tipos de feedback usando o site de feedback do Azure Maps.

## <a name="add-a-business-place-or-a-residential-address"></a>Adicione um local de negócios ou um endereço residencial 

Você pode querer fornecer feedback sobre um ponto de interesse ausente ou um endereço residencial. Há duas maneiras de fazer isso. Abra o site de feedback de dados do Mapa do Azure, procure as coordenadas do local perdido e clique em "Adicionar um lugar"

  ![procurar local ausente](./media/how-to-use-feedback-tool/search-poi.png)

Ou, você pode interagir com o mapa. Clique no local para soltar um pino na coordenada e clique em "Adicionar um lugar".

  ![adicionar pino](./media/how-to-use-feedback-tool/add-poi.png)

Ao clicar, você será direcionado a um formulário para fornecer os detalhes correspondentes para o local.

  ![adicionar um lugar](./media/how-to-use-feedback-tool/add-a-place.png)

## <a name="fix-a-business-place-or-a-residential-address"></a>Conserte um local de negócios ou um endereço residencial 

O site de feedback também permite pesquisar e localizar um local de negócios ou um endereço. Você pode fornecer feedback para corrigir o endereço ou o local do pino, se eles não estiverem corretos. Para fornecer feedback para corrigir o endereço, use a barra de pesquisa para procurar um local de negócios ou endereço residencial. Clique no local de seu interesse na lista de resultados. Clique em "Consertar este lugar".

  ![busca lugar para corrigir](./media/how-to-use-feedback-tool/fix-place.png)

Para fornecer feedback para corrigir o endereço, preencha o formulário "Corrigir um lugar" e clique no botão "enviar".

  ![forma de correção](./media/how-to-use-feedback-tool/fix-form.png)

Se o local do pino do local estiver errado, verifique a caixa de seleção no formulário "Corrigir um lugar" que diz "O local do pino está incorreto". Mova o pino para o local correto e clique no botão "enviar".

  ![mover localização do pino](./media/how-to-use-feedback-tool/move-pin.png)

## <a name="add-a-comment"></a>Adicionar um comentário 

Além de permitir que você procure por um local, a ferramenta de feedback também permite adicionar um comentário de texto de formulário gratuito para detalhes relacionados ao local. Para adicionar um comentário, procure o local ou clique no local. Clique em "Adicionar um comentário", escreva um comentário e clique em "Enviar".

  ![adicionar comentário](./media/how-to-use-feedback-tool/add-comment.png)

## <a name="track-status"></a>Rastreamento de status 

Você também pode acompanhar o status de sua solicitação verificando a caixa "Eu quero rastrear status" e fornecendo seu e-mail enquanto faz uma solicitação. Você receberá um link de rastreamento no e-mail que fornece um status atualizado de sua solicitação. 

  ![status de feedback](./media/how-to-use-feedback-tool/feedback-status.png)
