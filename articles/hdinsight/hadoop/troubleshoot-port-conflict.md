# <a name="scenario-port-conflict-when-starting-services-in-azure-hdinsight"></a>Cenário: Conflito portuário ao iniciar serviços no Azure HDInsight

Este artigo descreve etapas de solução de problemas e possíveis resoluções para problemas ao interagir com clusters Azure HDInsight.

## <a name="issue"></a>Problema

Um serviço não começa.

## <a name="cause"></a>Causa

Existe um conflito portuário.

## <a name="resolution"></a>Resolução

### <a name="method-1"></a>Método 1

Use os comandos abaixo para obter/matar todos os processos em execução, que são afetados pelo problema da porta.

```bash
netstat -lntp | grep <port>
ps -ef | grep <service>
kill -9 <service>
```

Então comece o serviço.

### <a name="method-2"></a>Método 2

Reinicie o nó.
