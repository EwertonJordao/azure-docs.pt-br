# <a name="azure-synapse-analytics-data-management-partners"></a>Parceiros de gerenciamento de dados do Azure Synapse Analytics

Este artigo destaca as empresas parceiras da Microsoft com ferramentas e soluções de gerenciamento de dados com suporte ao Azure Synapse.
