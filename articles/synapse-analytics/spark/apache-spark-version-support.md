# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>Versões de tempo de execução e idioma com suporte para Apache Spark e componentes dependentes  

O Apache Spark no serviço do Azure Synapse Analytics dá suporte a vários tempos de execução e serviços que este documento lista as versões.

## <a name="spark-version"></a>Versão do Spark

O Spark 2,4 é a versão com suporte no momento.

## <a name="operating-system-version"></a>Versão do sistema operacional

Apache Spark no Azure Synapse é executado no Ubuntu versão 16, 4.

## <a name="java-version"></a>Versão Java

Apache Spark no Azure Synapse inclui Java versão 1.8.0 _222

## <a name="scala-version"></a>Versão da escala

Apache Spark no Azure Synapse inclui escalabilidade 2.11.12

## <a name="net-core-version"></a>Versão do .NET Core

Apache Spark no Azure Synapse inclui o .NET Core 3,0

## <a name="net-support-for-apache-spark"></a>Suporte do .NET para Apache Spark

O Azure Synapse inclui suporte .NET para Apache Spark 0.5.0

## <a name="delta-lake"></a>Delta Lake

O Synapse Analytics inclui suporte para Delta Lake 0.5.0

## <a name="python-version"></a>Versão do Python

Apache Spark no Azure Synapse inclui uma instalação do Python versão 3.6.1
