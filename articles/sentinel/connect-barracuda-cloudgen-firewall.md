# <a name="connect-barracuda-cloudgen-firewall"></a>Conectar Firewall do Barracuda CloudGen

O conector Barracuda CloudGen Firewall (CGFW) permite conectar facilmente seus logs Barracuda CGFW com o Azure Sentinel, para visualizar dashboards, criar alertas personalizados e melhorar a investigação. Isso lhe dá mais informações sobre a rede da sua organização e melhora os recursos de operação de segurança.




## <a name="prerequisites"></a>Pré-requisitos

- Leia e escreva permissões para o espaço de trabalho do Azure Sentinel.

- Barracuda CloudGen Firewall deve ser configurado para exportar logs via Syslog.

## <a name="connect-azure-sentinel-to-barracuda-cloudgen-firewall"></a>Conecte o Azure Sentinel ao Barracuda CloudGen Firewall

1. No portal Azure, navegue até os**conectores** **azure Sentinel** > Data e selecione o conector **Barracuda CloudGen Firewall.**

2. Selecione **Abrir a página do conector**.

3. Siga as instruções na página **Barracuda CloudGen Firewall.**
