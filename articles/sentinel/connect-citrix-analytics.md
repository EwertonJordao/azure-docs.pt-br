# <a name="connect-citrix-analytics-security"></a>Conectar o Citrix Analytics (Segurança) 

A integração do Citrix Analytics (Security) com o Azure Sentinel ajuda você a exportar dados analisados para eventos de risco do Citrix Analytics (Security) para o Azure Sentinel. Você pode criar dashboards personalizados, analisar dados de outras fontes, juntamente com os do Citrix Analytics (Security) e criar fluxos de trabalho personalizados usando aplicativos lógicos para monitorar e mitigar eventos de segurança. 



## <a name="prerequisites"></a>Pré-requisitos

- Leia e escreva permissões para o espaço de trabalho do Azure Sentinel.

- Direito ao Citrix Security Analytics em Citrix Cloud.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Conecte o Azure Sentinel ao Citrix Analytics (Security)

1. No portal Azure, navegue até os**conectores** **azure Sentinel** > Data e selecione o conector **Citrix Analytics (Security).**

2. Selecione **Abrir a página do conector**.

3. Siga as instruções na página **Citrix Analytics (Security).**
