# <a name="connect-extrahop-revealx"></a>Conectar ExtraHop Reveal(x)

O conector de dados ExtraHop Reveal(x) permite conectar facilmente seu sistema Reveal(x) com o Azure Sentinel para visualizar dashboards, criar alertas personalizados e melhorar a investigação. Essa integração lhe dá a capacidade de obter insights sobre a rede da sua organização e melhorar os recursos de operação de segurança.


## <a name="prerequisites"></a>Pré-requisitos

- Leia e escreva permissões para o espaço de trabalho do Azure Sentinel.

- O aparelho ExtraHop Discover ou Command com firmware versão 7.8 ou posterior com uma conta de usuário que tenha privilégios ilimitados (administrador).

## <a name="connect-azure-sentinel-to-extrahop-revealx"></a>Conecte o Azure Sentinel ao ExtraHop Reveal(x)

1. No portal Azure, navegue até os**conectores** **Azure Sentinel** > Data e selecione o conector **ExtraHop Reveal(x).**

2. Selecione **Abrir a página do conector**.

3. Siga as instruções na página **ExtraHop Reveal(x).**
