# <a name="build-real-time-applications-with-azure-signalr-service"></a>Crie aplicativos em tempo real com o Azure SignalR Service

Com [o Azure SignalR Service,](https://azure.microsoft.com/services/signalr-service/)você pode facilmente adicionar funcionalidade em tempo real aos aplicativos. Essa funcionalidade em tempo real permite que o serviço empurre atualizações de conteúdo para aplicativos móveis conectados. Como resultado, os aplicativos são atualizados sem a necessidade de sondar o servidor ou enviar novas solicitações HTTP para atualizações.

## <a name="azure-signalr-service-features"></a>Recursos do Azure SignalR Service
- Possui amplo suporte ao cliente para uma ampla gama de clientes, como navegadores web e móveis, aplicativos de desktop, processo de servidor, dispositivos IoT e consoles de jogos.
- Permite que várias instâncias trabalhem juntas para dimensionar para milhões de conexões de aplicativos móveis. Ele também suporta várias regiões globais para fins de fragmentação, alta disponibilidade ou recuperação de desastres.
- Oferece APIs ricas para diferentes padrões de mensagens para enviar mensagens para uma determinada conexão, todas as conexões ou um subconjunto de conexões que pertencem a um usuário específico ou foram colocadas em um grupo arbitrário.
