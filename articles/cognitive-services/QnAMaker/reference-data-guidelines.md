# <a name="import-and-export-data-reference"></a>Referência de dados de importação e exportação

Revise esta referência de importação e exportação para obter os melhores resultados para sua base de conhecimento de backup, armazenamento e substituição.

## <a name="import-and-export-knowledge-base"></a>Base de conhecimento de importação e exportação

**Os arquivos TSV e XLS,** a partir de bases de conhecimento exportadas, só podem ser usados importando os arquivos da página **Configurações** no portal QnA Maker. Eles não podem ser usados como fontes de dados durante a criação da base de conhecimento ou a partir do **+ Adicionar arquivo** ou + Adicionar recurso **URL** na página **Configurações.**
