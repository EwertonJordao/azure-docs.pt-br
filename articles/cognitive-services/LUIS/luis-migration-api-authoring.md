# <a name="authoring-api-changes"></a>Alterações da API de autoria 

As APIs de autoria são atualizadas, para V3, para permitir a [decomposição do modelo com entidades aprendidas por máquina.](luis-concept-data-extraction.md) 

Você pode escrever no V2 ou visualizar V3 APIS. O mesmo aplicativo também pode optar por usar respostas de ponto final de previsão V2 ou V3. Você não precisa ter a mesma versão da API para autoria e resposta de ponto final de previsão.

O portal LUIS utiliza exclusivamente as APIs de autoria v3.

## <a name="next-steps"></a>Próximas etapas

Revisar [alterações no ponto final de previsão](luis-migration-api-v3.md) para API V3.
