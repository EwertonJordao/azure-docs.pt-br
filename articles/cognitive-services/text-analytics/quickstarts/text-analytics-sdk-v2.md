# <a name="quickstart-use-the-text-analytics-client-library-v2"></a>Início Rápido: Usar a biblioteca de clientes da Análise de Texto v2

Use a biblioteca de clientes da Análise de Texto v2 para executar:

* Análise de sentimento
* Detecção de idioma
* Reconhecimento de entidade
