## <a name="reference-docs"></a>Documentos de Referência

O serviço Speech fornece dois SDKs. O primeiro SDK é o [SDK](../speech-sdk.md) de fala principal e fornece a maioria das funcionalidades necessárias para interagir com o serviço Speech. O segundo SDK é específico para dispositivos, apropriadamente chamado [de Speech Devices SDK](../speech-devices-sdk.md). Ambos os SDKs estão disponíveis em muitos idiomas.

> O Serviço de Fala SDK é mantido e atualizado ativamente. Para acompanhar as alterações, as atualizações e as adições de recursos, consulte as [notas de versão do Speech SDK](../releasenotes.md).

### <a name="speech-devices-sdk-reference-docs"></a>Dispositivos de fala SDK médicos de referência

O [Speech Devices SDK](../speech-devices-sdk.md) é um superconjunto do Speech SDK, com funcionalidade estendida para dispositivos específicos. Para baixar o Speech Devices SDK, você deve primeiro [escolher um kit de desenvolvimento](../get-speech-devices-sdk.md#choose-a-development-kit).
