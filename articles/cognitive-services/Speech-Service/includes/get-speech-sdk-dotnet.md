#### <a name="c-nuget-package"></a>Pacote NuGet do C#

O SDK de fala do .NET pode ser instalado **.NET Core CLI** por meio do CLI do .NET Core `dotnet add` com o comando a seguir.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

O SDK de fala do .NET pode ser instalado por meio do **Gerenciador** de `Install-Package` pacotes com o comando a seguir.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
