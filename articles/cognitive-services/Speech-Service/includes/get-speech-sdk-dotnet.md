#### <a name="c-nuget-package"></a>C# NuGet Pacote

O .NET Speech SDK pode ser instalado a partir `dotnet add` do **.NET Core CLI** com o seguinte comando.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

O .NET Speech SDK pode ser instalado no `Install-Package` Gerenciador de **pacotes** com o seguinte comando.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
