Neste guia de início rápido, você usa a CLI de Fala na linha de comando para reconhecer a fala registrada em um arquivo de áudio e produzir uma transcrição de texto. É fácil usar a CLI de Fala para executar tarefas comuns de reconhecimento como transcrição de conversas. Após configurar uma única vez, a CLI de Fala permitirá transcrever o áudio em texto interativamente com um microfone ou em arquivos usando um script em lote.
