Este guia mostra como instalar o [SDK de Fala](~/articles/cognitive-services/speech-service/speech-sdk.md) para JavaScript para uso com o NodeJS. Se você quiser apenas o nome do pacote para começar por conta própria, execute `npm install microsoft-cognitiveservices-speech-sdk` de seu projeto existente.

[!INCLUDE [License Notice](~/includes/cognitive-services-speech-service-license-notice.md)]

## <a name="prerequisites"></a>Pré-requisitos

Este início rápido requer:

* [NodeJS](https://nodejs.org/)

## <a name="create-a-new-npm-project"></a>Crie um projeto NPM.

Crie um diretório, execute `npm init` e percorra seus prompts.

## <a name="add-the-speech-sdk-package"></a>Adicionar o pacote do SDK de Fala

Execute `npm install microsoft-cognitiveservices-speech-sdk`
