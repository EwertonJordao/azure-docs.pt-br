# <a name="sapi-phone-set-usage"></a>Uso do conjunto de telefones SAPI

A Microsoft define o conjunto de telefones SAPI para sete idiomas (en-US, fr-FR, de-DE, es-ES, ja-JP, zh-CN, zh-TW). O conjunto de telefones SAPI é relativamente fácil de usar e lembrar especialmente para o orador nativo. Por exemplo, ja-JP use kana como telefone, zh-CN use PinYin como conjunto de telefone. Mas, às vezes, não é eficiente o suficiente, por exemplo, o conjunto de telefones SAPI de fr-FR não podia representar a ligação francesa.
