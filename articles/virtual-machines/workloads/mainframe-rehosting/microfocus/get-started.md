# <a name="micro-focus-devtest-environments-on-azure"></a>Micro Focus dev/test environments no Azure

Começar no Azure com um ambiente de desenvolvimento e teste é uma ótima maneira de impulsionar seus esforços de modernização. Os aplicativos de mainframe que dependem de sistemas legados, como o IBM CICS (Customer Information Control System) e o IMS (Information Management System) podem ser rehospedados no Azure usando soluções de parceiros.

A Micro Focus fez uma parceria com a Microsoft para oferecer ferramentas atraentes de migração e rehospedagem de mainframe. Você pode executar aplicativos de mainframe em cenários de desenvolvimento, teste e produção no Azure e na plataforma Microsoft Windows.

A Micro Focus oferece soluções de rehospedagem e de rehospedagem de testes de desenvolvimento que estendem os aplicativos de negócios COBOL e PL/I existentes para o Azure.
