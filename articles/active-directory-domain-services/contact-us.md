# <a name="azure-ad-domain-services---contact-us"></a>Azure AD Domain Services: entre em contato conosco

## <a name="feedback"></a>Comentários
Se você tiver feedback para o nosso serviço ou perguntas não técnicas, você pode compartilhar sua solicitação com o grupo de produtos Azure AD Domain Service enviando-nos um e-mail para: [Azure AD Domain Services Feedback](mailto:aaddsfb@microsoft.com). Os e-mails para este endereço são revisados. Os membros do grupo de produtos **podem** responder à solicitação de mais informações. As solicitações de suporte técnico a este endereço de e-mail não serão respondidas.

## <a name="technical-assistance"></a>Assistência técnica
Se você tiver problemas com seu domínio gerenciado, verifique se as etapas descritas no [Guia de solução de problemas](troubleshoot.md) resolvem o problema. Se você ainda estiver com problemas, entre no portal Azure. Abra a página inicial **do Azure AD Domain Services** e selecione Nova solicitação de **suporte** do painel de ação em Support + solução **de problemas**.

![nova solicitação de suporte](./media/contact-us/supportRequest.png) 

Preencha cada seção da solicitação de suporte com a maior precisão possível. Ao escolher um serviço, certifique-se de selecionar **O Azure Active Directory Domain Services (VM - Controladores de Domínio)** para encaminhar sua solicitação para o profissional de suporte adequado.

> [!IMPORTANT]
> Selecione Diretórios ativos do **Azure, domínios e objetos** das listas de serviços para obter ajuda na validação de domínio sustais com o Azure Active Directory.
>
> 

## <a name="feature-requests"></a>Solicitações de recursos
Você tem uma ideia de como podemos melhorar os serviços de domínio do Azure Active Directory? Gostaríamos de ouvir sobre isso. Vá aos Fóruns de Feedback do Azure, navegue até a categoria [Serviços de Domínio](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593) e compartilhe sua ideia. Os membros do grupo de produtos revisam essas solicitações.
