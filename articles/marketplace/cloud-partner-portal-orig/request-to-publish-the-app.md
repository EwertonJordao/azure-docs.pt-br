# <a name="request-to-publish-the-app"></a>Solicitação para publicar o aplicativo

![Clique para Publicar](./media/CRMScreenShot18.png)

1. Clique no botão Publicar para enviar sua oferta à nossa equipe de certificação. Você poderá conferir o status da oferta na guia Status.

![Examinar o status de publicação](./media/CRMScreenShot19.png)

## <a name="republish-the-app"></a>Republicar o aplicativo

 Se precisar fazer uma solicitação para republicar um aplicativo, primeiro clique no botão Cancelar Publicação na guia Status (se a solicitação não estiver mais ativa, ignore esta etapa). Faça as alterações no aplicativo, atualize devidamente a oferta e clique no botão Publicar para reenviar. # Considerações especiais para o Microsoft Dynamics 365 Business Edition, observe que as soluções criadas para as Business Editions do Microsoft Dynamics 365 passarão por verificações adicionais durante a certificação para garantir que os limites do Business Edition sejam atendidos. Os limites do Business Edition estão documentados aqui. # Obtenção de ajuda. Se você enfrentar problemas, registre um tíquete de suporte clicando em no canto superior direito e acessando.
