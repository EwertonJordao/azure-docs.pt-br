# <a name="power-bi-app-offer"></a>Oferta de aplicativo Power BI

O Portal do Cloud Partner não dá mais suporte à criação e ao gerenciamento de ofertas de aplicativos Power BI. Essa funcionalidade foi movida para o Microsoft [Partner Center](https://partner.microsoft.com/pcv/). Para obter mais informações, consulte [criar um aplicativo Power bi para Microsoft AppSource](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer).
