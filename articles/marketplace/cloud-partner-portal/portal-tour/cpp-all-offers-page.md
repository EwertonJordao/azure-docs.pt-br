# <a name="all-offers-page"></a>Página de todas as ofertas

A página [**Todas as ofertas**](https://cloudpartner.azure.com/#alloffers) permite que os parceiros exibam uma lista de ofertas criadas, incluindo seu nome, tipo, publicador e status atual.  Você pode especificar uma cadeia de caracteres para filtrar os resultados.   A área de status na parte inferior da página exibe as contagens de página e de agregação e permite que você navegue entre as páginas de lista. Ao selecionar uma oferta nesta lista, você exibirá essa oferta no editor de oferta associada.
