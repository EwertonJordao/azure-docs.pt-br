# <a name="manage-azure-and-appsource-marketplace-offers"></a>Gerenciar ofertas dos Marketplaces do Azure e do AppSource

Esta seção apresenta operações gerais usadas para gerenciar as ofertas para os Marketplaces do Azure e do AppSource.  Por exemplo, ela explica como usar o [Portal do Cloud Partner](https://cloudpartner.azure.com/) para criar e publicar uma oferta do marketplace.  Esta seção também explica os problemas técnicos que são comuns à maioria dos tipos de oferta, tais como ativos de imagem e com base em texto.


## <a name="standard-offer-operations"></a>Operações de oferta padrão

O Portal do Cloud Partner permite que um publicador execute as operações básicas a seguir em uma oferta.

|     Operação      |  Descrição                                           |
|     ---------      |  -----------                                           |
| [Criar oferta](./cpp-create-offer.md)   | Cria uma entrada de oferta pendente no repositório de integração, mas não publica-o automaticamente em um Microsoft Marketplace | 
| [Exibir status da oferta](./cpp-view-status-offer.md)   | Fornece o status da publicação de uma oferta.  Há vários locais no portal em que as informações de status são exibidas. |
| [Publicar oferta](./cpp-publish-offer.md) | Valida e, em seguida, posta uma entrada de oferta concluído no Microsoft Marketplace apropriado.  A oferta é então considerada *ativa*. |
| [Atualizar oferta](./cpp-update-offer.md)   | Permite a modificação de certas características de uma oferta existente.  Aplica-se às ofertas pendentes e publicadas. |
| [Excluir oferta](./cpp-delete-offer.md)   | Exclui uma oferta existente pendente ou ativa.  | 
|  |  |
