# <a name="azure-saas-application-offer"></a>Oferta de aplicativo de SaaS do Azure

O [Portal de Parceiros em Nuvem](https://cloudpartner.azure.com/) não suporta mais a criação e o gerenciamento de ofertas de aplicativos SaaS.  Em vez disso, essa funcionalidade foi movida para o portal microsoft [partner center](https://partner.microsoft.com/pcv/).  Para obter mais informações, consulte [Criar uma nova oferta SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Próximas etapas

Antes de criar uma nova oferta de aplicativo SaaS, você deve ler o [Guia de Publicação de Oferta de Aplicativos SaaS](../../marketplace-saas-applications-technical-publishing-guide.md) e rever as etapas do portal do Partner Center listadas na lista de [verificação de criação de ofertas](../../partner-center-portal/offer-creation-checklist.md).
