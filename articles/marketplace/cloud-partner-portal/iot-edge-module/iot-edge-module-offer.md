# <a name="iot-edge-module-offer"></a>Oferta de módulo do IoT Edge

O [portal de parceiros de nuvem](https://cloudpartner.azure.com/) não dá mais suporte à criação e ao gerenciamento de ofertas de módulo IOT Edge. Essa funcionalidade foi movida para o Microsoft [Partner Center](https://partner.microsoft.com/pcv/). Para obter mais informações, consulte [criar, configurar e publicar uma oferta de módulo IOT Edge no Azure Marketplace](../../partner-center-portal/azure-iot-edge-module-creation.md).
