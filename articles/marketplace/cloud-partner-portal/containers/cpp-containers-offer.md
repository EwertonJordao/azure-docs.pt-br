# <a name="containers"></a>Contêineres

O [Portal do Cloud Partner](https://cloudpartner.azure.com/) não dá mais suporte à criação e ao gerenciamento de ofertas de Contêiner do Azure. Essa funcionalidade foi movida para o Microsoft [Partner Center](https://partner.microsoft.com/pcv/). Para obter mais informações, confira [Criar uma oferta de Contêiner do Azure](../../partner-center-portal/create-azure-container-offer.md).
