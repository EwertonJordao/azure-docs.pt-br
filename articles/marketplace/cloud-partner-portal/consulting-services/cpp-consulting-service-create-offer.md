# <a name="create-a-consulting-offer"></a>Crie uma oferta de consultoria

>[!Important]
>A partir de 30 de março de 2020, começaremos a movimentar a gestão de suas ofertas de serviços de Consultoria para o Partner Center. Após a migração, você criará e gerenciará suas ofertas no Partner Center. Siga as instruções em [Visão geral de criação de serviços de consultoria](https://aka.ms/AzureCreateConsultingService) para gerenciar suas ofertas migradas.

Este artigo descreve como criar e publicar uma oferta de serviço de consultoria.

## <a name="define-your-offer"></a>Definir sua oferta

Defina sua oferta de serviços de consultoria em pacote. Concentre-se em escopo fixo, duração fixa, preço estimado, preço fixo (ou gratuito) e, principalmente, em ofertas orientadas à pré-venda para um único cliente. Selecione participações em pacotes repetíveis que são populares e eficazes para gerar novos negócios para você.

## <a name="create-a-new-offer"></a>Criar uma oferta

Para criar uma nova oferta, siga estas etapas.
1. No menu principal do Portal do Cloud Partner, selecione **Nova oferta**.
1. No menu **Nova oferta**, selecione **Serviço de consultoria**.

    ![Criar uma nova oferta de serviços de consultoria](media/cppselectnewconsultingoffer.png)
