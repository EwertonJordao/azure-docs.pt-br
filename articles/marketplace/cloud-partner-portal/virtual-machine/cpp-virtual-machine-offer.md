# <a name="azure-virtual-machine-offer"></a>Oferta de máquinas virtuais do Azure

[Portal do Cloud Partner](https://cloudpartner.azure.com/) não tem mais suporte para criação e gerenciamento de ofertas de máquina virtual do Azure. Essa funcionalidade foi movida para o Microsoft [Partner Center](https://partner.microsoft.com/pcv/). Para mais informações, veja [Criar uma oferta de máquina virtual do Azure](../../partner-center-portal/azure-vm-create-offer.md).
