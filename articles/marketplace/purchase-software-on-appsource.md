# <a name="how-to-purchase-software-on-microsoft-appsource"></a>Como comprar software no Microsoft AppSource

O Microsoft [AppSource](https://appsource.microsoft.com/) agora permite que os clientes assinem aplicativos SaaS oferecidos pelos parceiros da Microsoft. Os clientes podem encontrar aplicativos web certificados na loja e podem gerenciar as taxas, upgrades, downgrades e cancelamentos em um único lugar usando o Admin Center da Microsoft. Este artigo descreve como você pode comprar um aplicativo na loja.

O AppSource fornece uma seleção de aplicativos para escolher, incluindo testes gratuitos e assinaturas pagas. Este artigo descreve como as assinaturas pagas podem ser compradas via AppSource.

## <a name="requirements"></a>Requisitos

Para comprar ofertas SaaS, você precisa:

- Uma conta corporativa ou de estudante. Se você não tem uma conta de trabalho ou escola, siga o link para "Cadastre-se para uma conta gratuita" durante o processo de compra.

- Um cartão de crédito válido no país/região onde você está comprando a oferta.

    > [!Note]
    > O país de faturamento de cartão de crédito pode ser bloqueado na localização da sua instância do Azure Active Directory.

## <a name="steps-for-making-purchases-on-appsource"></a>Etapas para fazer compras no AppSource

1. Selecione sua região de faturamento/país no [AppSource](https://appsource.microsoft.com/).
1. Encontre o aplicativo que deseja comprar.
1. Selecione **Obtê-lo agora**.
1. Escolha o plano:

    1. Defina o número de assentos
    1. Defina o prazo de faturamento.
    
1. Digite seu endereço de cobrança.
1. Digite seu método de pagamento:
    * Cartões de crédito suportados: Visa, Master Card, American Express, Discover.
    
1. Selecione **O pedido de lugar**.

## <a name="how-to-configure-software-post-purchase"></a>Como configurar software pós-compra

Depois que seu pedido é recebido, pode levar vários segundos para ser confirmado. Você receberá um link para configurar sua assinatura SaaS na página, bem como um e-mail confirmando a compra e o link para concluir a configuração.

1. Na página **Confirmação de pedido,** **selecione Configurar**. Alternativamente, você pode usar o link configuração enviado para você no e-mail de confirmação.
1. Provisionamento completo e configuração no site do provedor de software.

    > [!Note]
    > Esta etapa é obrigatória e pode ser concluída a qualquer momento.
