# <a name="downloads-dashboard-in-commercial-marketplace-analytics"></a>Downloads Dashboard em análises de mercado comercial

Este artigo fornece informações sobre o painel Downloads no Partner Center. Este painel exibe uma lista de suas solicitações de download nos últimos 30 dias.

Para acessar o painel Downloads, abra o painel **[Analisar](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)** em Comercial Marketplace.

>[!NOTE]
> Para definições detalhadas da terminologia de análise, consulte [perguntas freqüentes e terminologia para análises de mercado comercial](./faq-terminology.md).

## <a name="downloads-dashboard"></a>Painel de downloads

O painel **Downloads** do menu **Analisar** exibe solicitações para quaisquer downloads que contenham mais de 1000 linhas de dados de clientes ou pedidos.

Você receberá uma notificação pop-up e um e-mail contendo um link para o painel **Downloads** sempre que solicitar um download com mais de 1000 linhas de dados. Esses downloads de dados estarão disponíveis por um período de 30 dias e depois removidos.
