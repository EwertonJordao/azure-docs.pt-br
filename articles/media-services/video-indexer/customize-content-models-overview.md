# <a name="customizing-content-models-in-video-indexer"></a>Personalização de modelos de conteúdo no Indexador de Vídeo

O Video Indexer permite que você personalize alguns de seus modelos para serem adaptados ao seu caso de uso específico. Esses modelos incluem [marcas,](customize-brands-model-overview.md) [linguagem](customize-language-model-overview.md)e [pessoa.](customize-person-model-overview.md) Você pode personalizar facilmente esses modelos usando o site ou API do Video Indexer.

Este artigo fornece links para artigos que explicam os benefícios de cada tipo de personalização. O artigo também se vincula a guias de como fazer que mostram como você pode implementar a personalização de cada modelo.
