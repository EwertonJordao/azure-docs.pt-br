# <a name="azure-media-services-v3-documentation-updates"></a>Atualizações de documentação do Azure Media Services v3

Este artigo fala sobre as mais recentes atualizações de documentação do Media Services v3.

## <a name="april-2020"></a>Abril de 2020

* Os documentos do Azure Media Player foram migrados para a documentação do [Azure.](../azure-media-player/azure-media-player-overview.md)
* A [transmissão ao vivo com o Open Broadcasting Studio (OBS)](live-events-obs-quickstart.md) quickstart foi adicionada. Ele mostra como criar uma transmissão ao vivo do Media Services usando o portal Azure e o OBS.
