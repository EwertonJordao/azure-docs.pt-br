# <a name="get-started-with-azure-security-center-for-iot"></a>Introdução à Central de Segurança do Azure para IoT

Este artigo fornece uma explicação dos diferentes componentes da central de segurança do Azure para o serviço de IoT e explica como começar a usar o serviço usando duas opções de implantação possíveis.

## <a name="deployment-options"></a>Opções de implantação

Escolha o cenário de serviço que melhor atenda aos requisitos de dispositivo e de ambiente de IoT.

### <a name="built-in-deployment"></a>Implantação interna

Usando a opção de implantação integrada, integrada, a central de segurança do Azure para IoT pode ser rapidamente integrada ao seu hub IoT e fornecer análise de segurança da configuração do Hub IoT, identidade e gerenciamento do dispositivo e padrões de comunicação do dispositivo Hub.

Inicie uma [implantação interna com](iot-hub-integration.md) recomendações e monitoramento do Hub IOT.
    <br>

### <a name="enhanced-deployment"></a>Implantação avançada

Para recursos de segurança aprimorados, implantar a central de segurança do Azure para agentes de IoT, além de habilitar a segurança do Hub IoT, fornece coleta de eventos baseada em agente, análise e detecção de ameaças de dados de segurança chave de seus dispositivos IoT, bem como recursos de gerenciamento de postura de segurança abrangentes.

Inicie uma [implantação avançada](security-agents.md) com uma solução de gerenciamento de postura de segurança e proteção de ameaças abrangente baseada em agente.
