# <a name="get-started-with-azure-security-center-for-iot"></a>Introdução à Central de Segurança do Azure para IoT

Este artigo fornece uma explicação dos diferentes blocos de construção do Azure Security Center para serviço de IoT e explica como começar com o serviço usando duas opções de implantação possíveis.  

## <a name="deployment-options"></a>Opções de implantação

Escolha o cenário de serviço que melhor atende aos seus requisitos de dispositivo e ambiente IoT. 

### <a name="built-in-deployment"></a>Implantação incorporada
Usando a opção de implantação integrada e incorporada, o Azure Security Center for IoT pode ser rapidamente integrado ao seu Hub ioT e fornecer análises de segurança da configuração do hub IoT, identidade e gerenciamento de dispositivos e padrões de comunicação de dispositivos de hub.

Inicie [uma implantação incorporada](iot-hub-integration.md) com monitoramento e recomendações do IoT Hub. 
    <br>

### <a name="enhanced-deployment"></a>Implantação avançada
Para recursos de segurança aprimorados, a implantação do Azure Security Center para agentes de IoT, além de permitir a segurança do IoT Hub, fornece coleta, análise e detecção de ameaças baseadas em agentes de coleta, análise e detecção de ameaças de dados de segurança importantes de seus dispositivos IoT, bem como abrangente recursos de gerenciamento de postura de segurança.

Inicie uma [implantação aprimorada](security-agents.md) com uma solução abrangente de proteção contra ameaças e gerenciamento de postura de segurança baseada em agentes.
