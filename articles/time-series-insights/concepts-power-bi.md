# <a name="connect-time-series-insights-to-power-bi"></a>Conecte insights da série de tempo ao POWER BI  

Você está procurando uma solução para ajudar a correlacionar seus dados da série temporal, criar visualizações vívidas e compartilhar insights em toda a sua organização?

O Azure Time Series Insights agora se integra perfeitamente ao [Power BI,](https://powerbi.microsoft.com/)fornecendo recursos de visualização e dashboarding mais poderosos sobre seus dados de streaming e permite que você compartilhe insights e resultados em toda a sua organização.

Ao conectar seus dados ao Power BI, agora você pode:

* Realize correlações com outras fontes de dados suportadas pelo Power BI e acesse uma série de diferentes opções de visualização de dados. 
* Crie dashboards e relatórios do Power BI usando seus dados do Time Series Insight e compartilhe-os com sua organização. 
* Desbloqueie cenários de interop de dados de forma simples e fácil de usar e obtenha insights mais rápido do que nunca.
* Modifique os dados do Time Series Insights no Power BI usando o poderoso Editor Avançado.

## <a name="native-connector"></a>Conector nativo

Conectar seus dados leva menos de um minuto. Ao exportar rapidamente sua visão do Time Series Insight Explorer para o Power BI, qualquer pessoa em sua organização pode acessar e transformar dados de forma intuitiva e visual.

As opções de configuração incluem escolher entre dados de nível **Raw** e **Agregado** e selecionar um intervalo de tempo desejado. 

Uma vez conectados os dados, todas as ferramentas de visualização e gráficos do Power BI estão disponíveis para melhorar ainda mais seus recursos de compartilhamento de informações. 

   [![Exemplos de gráficos no Power BI](./media/concepts-connect-power-bi/power-bi-tsi-example.png)](./media/concepts-connect-power-bi/power-bi-tsi-example.png#lightbox)

Leia [como conectar o Time Series Insights ao Power BI](./how-to-connect-power-bi.md) para obter detalhes passo a passo.

## <a name="flexible-workflow"></a>Fluxo de trabalho flexível
 
Use o conector de qualquer lugar do seu fluxo de trabalho:

* Se você estiver no Time Series Insights Explorer, você pode exportar rapidamente sua exibição para o Power BI. 
* Se você já estiver no Power BI, aumente seus cenários de visualização existentes usando dados de consulta do Time Series Insights.

Para consultar e editar a funcionalidade avançada dentro do Power BI, use o Editor Avançado do Power BI.
