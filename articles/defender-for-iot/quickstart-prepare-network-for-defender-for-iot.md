# <a name="quickstart-prepare-your-network-for-azure-defender-for-iot-deployment"></a>Início Rápido: Preparar sua rede para a implantação do Azure Defender para IoT

## <a name="choose-your-hardware"></a>Escolher o hardware
O Azure Defender para IoT dá suporte a vários dispositivos certificados, incluindo:
- Dispositivos pré-configurados certificados, nos quais o software já está instalado
- Dispositivos certificados não configurados nos quais você pode baixar e instalar o software do sensor
- Especificações do dispositivo do console de gerenciamento local. O console de gerenciamento local não está disponível como um dispositivo pré-configurado. 

Para obter mais informações, confira o [Guia de Especificações de Hardware](https://aka.ms/AzureDefenderforIoTBareMetalAppliance)


## <a name="install-components"></a>Instalar componentes
Depois de adquirir os componentes de hardware e baixar seu pacote de software do portal do Azure Defender para IoT, você poderá instalar o software de componente. Para obter mais informações, confira o [Guia de Instalação](https://aka.ms/AzureDefenderforIoTInstallSensorISO).

## <a name="onboard-sensors-to-the-azure-defender-for-iot-portal"></a>Integrar sensores ao portal do Azure Defender para IoT 
 Este artigo fornece uma visão geral dos processos de implantação, integração e atualização do console de gerenciamento local e dos sensores do Azure Defender para IoT. Esse processo é executado seguindo as instruções descritas na página Bem-vindo(a) ao Azure Defender para IoT e as páginas seguintes.

![Captura de tela da exibição de página inicial do Azure](media/updates/image4.png)

Para abrir a página inicial:

1. Faça logon na sua conta do Azure e procure Azure Defender para IoT.

1. A **implantação do sensor** exige que você: Compre um hardware pré-configurado da Arrow ou compre um dispositivo próprio e instale o software do sensor nele.
1. A **implantação do console de gerenciamento local** exige que você compre um dispositivo próprio e instale o software do console de gerenciamento local nele. Para obter mais informações.




## <a name="network-setup"></a>Configuração da rede
Depois de instalar os componentes do sistema do Defender para IoT, você poderá configurar seu equipamento de rede para o monitoramento de rede ideal. Para obter mais informações, confira o [Guia de configuração de rede](https://aka.ms/AzureDefenderForIoTNetworkSetup).
