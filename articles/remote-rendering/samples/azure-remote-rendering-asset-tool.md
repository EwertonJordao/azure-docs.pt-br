# <a name="azure-remote-rendering-asset-tool-arrt"></a>ARRT (Azure Remote Rendering Asset Tool)

O ARRT (Azure Remote Rendering Asset Tool) é um aplicativo de área de trabalho de software livre desenvolvido em C++/Qt.

![ARRT](./media/azure-remote-rendering-asset-tool.png "Captura de tela do ARRT")

## <a name="features"></a>Recursos

O aplicativo pode ser usado para:

* Fazer upload de um modelo 3D
* Controlar a conversão de modelo
* Criar e gerenciar uma sessão de renderização remota
* Carregar um modelo 3D
* Visualizar o modelo 3D
* Modificar seus materiais

Você também pode usá-lo como um exemplo de software livre para saber como implementar um front-end para o SDK C++ do Azure Remote Rendering, usando a Biblioteca de Clientes do Armazenamento do Azure para gerenciar a conversão do modelo 3D.

## <a name="source-repository"></a>Repositório de origem

Você pode encontrar o código-fonte e a documentação no [repositório do ARRT no GitHub](https://github.com/Azure/azure-remote-rendering-asset-tool).
