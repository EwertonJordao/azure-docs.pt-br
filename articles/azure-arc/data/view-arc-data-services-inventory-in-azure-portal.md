# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Exibir o inventário de suas instâncias no portal do Azure

Depois de carregar suas [métricas, logs](upload-metrics-and-logs-to-azure-monitor.md)ou [uso](view-billing-data-in-azure.md), você poderá exibir sua instância como um recurso do Azure do portal do Azure. Para fazer isso, navegue até a URL especial [https://portal.azure.com](https://portal.azure.com) e siga as capturas de tela abaixo

1. Ir para todos os serviços
1. Pesquisar o tipo de tipos de instâncias de banco de dados
1. Adicione-o aos seus favoritos
1. Clique no tipo de instância na barra de navegação à esquerda
1. Exiba suas instâncias na mesma exibição que seus outros recursos de hiperescala do Azure SQL ou PostgreSQL do Azure (Use filtros para obter uma exibição granular)
