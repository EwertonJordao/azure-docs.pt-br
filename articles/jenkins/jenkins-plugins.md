# <a name="jenkins-plugins-for-azure"></a>Plugins Jenkins para Azure

Os seguintes plugins Jenkins suportam vários recursos para uso com o Azure.
  
| Plugin Jenkins | Descrição                                   |
|------------------------------------------------------------------------------|
| [Plugin azure App Service](https://plugins.jenkins.io/azure-app-service)     | O plugin Jenkins para implantar um Azure App Service (atualmente suporta apenas o Web App). | 
| [Plug-in do Azure AD](https://plugins.jenkins.io/azure-ad)                       | Plugin Jenkins que suporta autenticação & autorização via Azure Active Directory. | 
| [Plugin Azure Artifact Manager](https://plugins.jenkins.io/azure-artifact-manager) | O plugin Azure Artifact Manager é um Gerenciador de Artefatos que permite armazenar seus artefatos no Azure Blob Storage. O plugin Azure Artifact Manager funciona de forma transparente para Jenkins e seus trabalhos, é como o gerenciador de artefatos padrão. | 
| [Plug-in de Agentes do Contêiner do Azure](https://plugins.jenkins.io/azure-container-agents) | O plugin Azure Container Agents pode ajudá-lo a executar um contêiner como um agente em Jenkins | 
| [Plugin de tarefa de registro de contêiner do Azure](https://plugins.jenkins.io/azure-container-registry-tasks)       | Plugin Jenkins para enviar uma solicitação de compilação de dockers ao [Registro de Contêineres do Azure](/azure/container-registry/container-registry-tasks-overview). |
| [Plug-in do Serviço de Contêiner do Azure](https://plugins.jenkins.io/azure-acs)       | Plugin Jenkins para implantar configurações no Azure Container Service (AKS). | 
| [Plug-in de Credencial do Azure](https://plugins.jenkins.io/azure-credentials)      | Plugin Jenkins para gerenciar as credenciais do Azure. | 
| [Plug-in da Função do Azure](https://plugins.jenkins.io/azure-function)           | Plugin Jenkins para implantar uma função Azure. | 
| [Plugin Azure IoT Edge](https://plugins.jenkins.io/azure-iot-edge)           | O plugin Azure IoT Edge facilita a configuração de um pipeline ci/cd para o desenvolvimento de uma solução IoT Edge em Jenkins. | 
| [Plug-in do Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Plugin Jenkins para projetos linux Azure Service Fabric. |
| [Plug-in de Armazenamento do Azure](https://plugins.jenkins.io/windows-azure-storage)     | plugin para upload de artefatos de construção ou download de dependências de compilação do armazenamento Microsoft Azure Blob. | 
| [Plugin de agentes azure VM](https://plugins.jenkins.io/azure-vm-agents)         | Plugin Jenkins para criar agentes Jenkins em máquinas virtuais Azure (via modelo Azure Resource Manager). | 
| [Plugin de conjunto de escala de máquina virtual do Azure](https://plugins.jenkins.io/azure-vmss)           | Plugin Jenkins para implantar imagens VM em conjuntos de escala de máquinas virtuais do Azure. |
