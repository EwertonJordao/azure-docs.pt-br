# <a name="activity-management-overview"></a>Visão geral do gerenciamento de atividades

O CloudSimple acompanha toda a atividade que pode afetar o funcionamento do seu ambiente private cloud. As atividades incluem alertas, eventos, tarefas e atividades de auditoria. As [páginas Atividade](monitor-activity.md) resumem todas as atividades atuais e permitem que você detalhes adicionais.

## <a name="events"></a>Eventos

Eventos acompanham a atividade do usuário e do sistema no portal CloudSimple. Os eventos mostram a atividade associada a um recurso específico e a gravidade do impacto.  Você pode visualizar os eventos a partir do portal CloudSimple.

## <a name="alerts"></a>Alertas

Alertas são notificações de qualquer atividade significativa em seu ambiente CloudSimple. Eventos que afetam o faturamento ou o acesso do usuário são mostrados como alertas.  Você pode reconhecer alertas do portal CloudSimple.

## <a name="tasks"></a>Tarefas

As tarefas rastreiam qualquer operação do usuário que leve mais de 30 segundos para ser concluída. Você pode monitorar o progresso de uma tarefa a partir do portal CloudSimple.  Para tarefas concluídas, as informações incluem o tempo total para conclusão.

## <a name="audit"></a>Audit

Os registros de auditoria acompanham as operações do usuário. Os registros de auditoria contêm os parâmetros fornecidos para a operação pelo usuário.  Você pode usar registros de auditoria para monitorar a atividade do usuário para todos os usuários.
