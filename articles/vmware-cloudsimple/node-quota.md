# <a name="cloudsimple-node-quota-limits"></a>Limites de cotas de nó cloudSimple

Quatro nós é a quantidade padrão disponível para compra quando sua assinatura está habilitada para o serviço CloudSimple.  Você pode comprar qualquer tipo de [nó](cloudsimple-node.md) no portal Azure.  Pelo menos três nós do mesmo SKU são necessários para criar uma Nuvem Privada.  Se você comprou os nós, você pode ver um erro ao tentar comprar nomes adicionais.

## <a name="quota-increase"></a>Aumento da cota

Você pode aumentar a cota do nó enviando uma solicitação de suporte. A equipe de operações de serviço avaliará a solicitação e trabalhará com você para aumentar a cota de nó.  Selecione as seguintes opções ao abrir um novo bilhete:

* Tipo de emissão: **Técnico**
* Assinatura: **Seu ID de assinatura**
* Tipo de serviço: **Solução VMware por CloudSimple**
* Tipo de **problema: Cota de ádes dedicados**
* Subtipo do problema: **Aumentar a cota de nódulos dedicados**
* Assunto: **Aumento de cotas**

Nos detalhes do bilhete de suporte, forneça o número necessário de nós e nó SKU.

* Nó SKU
* Número de nódulos adicionais para os quais você está solicitando o aumento da cota
