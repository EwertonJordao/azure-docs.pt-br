# <a name="avs-node-quota-limits"></a>Limites de cota do nó AVS

Quatro nós é a quantidade padrão disponível para compra quando sua assinatura está habilitada para o serviço AVS. Você pode adquirir qualquer [tipo de nó](cloudsimple-node.md) do portal do Azure. Pelo menos três nós do mesmo SKU são necessários para criar uma nuvem privada de AVS. Se você comprou os nós, poderá ver um erro ao tentar comprar nós adicionais.

## <a name="quota-increase"></a>Aumento da cota

Você pode aumentar a cota do nó enviando uma solicitação de suporte. A equipe de operações de serviço avaliará a solicitação e trabalhará com você para aumentar a cota do nó. Selecione as seguintes opções ao abrir um novo tíquete:

* Tipo de problema: **técnico**
* Assinatura: **sua ID de assinatura**
* Tipo de serviço: **solução VMware por AVS**
* Tipo de problema: **cota de nós dedicados**
* Subtipo de problema: **aumentar a cota de nós dedicados**
* Assunto: **aumento de cota**

Nos detalhes do tíquete de suporte, forneça o número necessário de nós e SKU do nó.

* SKU do nó
* Número de nós adicionais para os quais você está solicitando o aumento de cota
