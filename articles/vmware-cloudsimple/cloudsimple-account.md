# <a name="account-management-overview"></a>Visão geral do gerenciamento de contas

Quando você cria seu serviço CloudSimple, ele cria uma conta no CloudSimple.  A conta está associada à sua assinatura do Azure onde o serviço está localizado.  Todos os usuários com **funções de proprietário** e **contribuinte** na assinatura têm acesso ao portal CloudSimple.  O ID de assinatura do Azure e o ID do inquilino associados ao serviço CloudSimple são encontrados na [página Contas](account.md).

## <a name="additional-alert-emails"></a>E-mails de alerta adicionais

Você pode configurar IDs de e-mail no CloudSimple para receber alertas:

* Relacionado ao seu serviço
* Para processamento automático

## <a name="cloudsimple-operator-access"></a>Acesso ao operador CloudSimple

Você pode controlar o acesso ao portal CloudSimple para o pessoal de operações de serviço.  O pessoal de operações de serviço entra no portal quando você envia um bilhete de suporte.  As operações de serviço corrigirão quaisquer problemas relatados e as ações tomadas estão disponíveis nos registros de auditoria.

## <a name="users"></a>Usuários

Todos os usuários que possuem **função de proprietário** e **contribuinte** na assinatura têm acesso ao portal CloudSimple.  Quando você acessa o portal, o usuário é criado na conta CloudSimple.  Você pode desativar o acesso ao portal CloudSimple para usuários específicos a partir da página Contas.
