### <a name="requirements"></a>Requisitos

* Se você optar por usar o PowerShell localmente:
  * [Instale o módulo AZ PowerShell](/powershell/azure/install-az-ps).
  * Conecte-se à sua conta do Azure usando o cmdlet [Connect-AzAccount](/powershell/module/az.accounts/connect-azaccount) .
* Se você optar por usar Cloud Shell:
  * Consulte [visão geral do Azure cloud Shell](https://docs.microsoft.com/azure/cloud-shell/overview) para obter mais informações.
