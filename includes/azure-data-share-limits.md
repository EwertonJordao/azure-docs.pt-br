O Azure Data Share permite que as organizações compartilhem dados de forma simples e segura com seus clientes e parceiros.

| **Recurso** | **Limite** |
| --- | --- |
| Número máximo de recursos de compartilhamento de dados por assinatura do Azure |50 |
| Número máximo de ações enviadas por recurso data share |100 |
| Número máximo de ações recebidas por recurso data share |100 |
| Número máximo de convites por ação enviada |100 |
| Número máximo de assinaturas de ações por ação enviada |100 |
| Número máximo de conjuntos de dados por compartilhamento |100 |
| Número máximo de horários de instantâneos por ação |1 |
