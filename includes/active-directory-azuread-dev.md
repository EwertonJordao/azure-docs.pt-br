> [!IMPORTANT]
> A [plataforma de identidade da Microsoft (v 2.0)](/azure/active-directory/develop/v2-overview) é uma evolução da plataforma de desenvolvedor do Azure Active Directory (AD do Azure) (v 1.0). Ela permite que os desenvolvedores criem aplicativos que se conectam a todas as identidades da Microsoft e obtêm tokens para chamar APIs da Microsoft, tais como o Microsoft Graph ou APIs que os desenvolvedores criaram.
> Esse conteúdo é para o ponto de extremidade do Azure AD v 1.0 mais antigo. Recomendamos que você use o ponto de extremidade v2.0 para novos projetos. Para obter mais informações, leia [por que atualizar para a plataforma Microsoft Identity (v 2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) bem como [limitações da plataforma de identidade da Microsoft](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations).
