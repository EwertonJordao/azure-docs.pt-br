> [!IMPORTANT]
> [A plataforma de identidade microsoft (v2.0)](/azure/active-directory/develop/v2-overview) é uma evolução da plataforma de desenvolvedores Azure Active Directory (Azure AD) (v1.0). Ela permite que os desenvolvedores criem aplicativos que se conectam a todas as identidades da Microsoft e obtêm tokens para chamar APIs da Microsoft, tais como o Microsoft Graph ou APIs que os desenvolvedores criaram.
> Este conteúdo é para o ponto final mais antigo, Azure AD v1.0. Recomendamos que você use o ponto de extremidade v2.0 para novos projetos. Para obter mais informações, leia [Por que atualizar para a plataforma de identidade da Microsoft (v2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) bem como [as limitações da plataforma de identidade da Microsoft.](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations)
