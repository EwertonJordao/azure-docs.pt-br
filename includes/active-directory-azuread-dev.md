> [!IMPORTANT]
> A [plataforma de identidade da Microsoft (v2.0)](/azure/active-directory/develop/v2-overview) é uma [evolução da plataforma de desenvolvedor (v1.0) do Azure AD (Azure Active Directory)](../articles/active-directory/azuread-dev/about-microsoft-identity-platform.md). Ela permite que os desenvolvedores criem aplicativos que se conectam a todas as identidades da Microsoft e obtêm tokens para chamar APIs da Microsoft, tais como o Microsoft Graph ou APIs que os desenvolvedores criaram.
> Este conteúdo é para o ponto de extremidade do Azure AD v1.0 mais antigo. Recomendamos que você use o ponto de extremidade v2.0 para novos projetos. Para obter mais informações, confira [Por que atualizar para a plataforma de identidade da Microsoft (v2.0)?](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison) e também [limitações da plataforma de identidade da Microsoft](/azure/active-directory/azuread-dev/azure-ad-endpoint-comparison#limitations).
