O Azure Database Migration Service é um serviço totalmente gerenciado projetado para permitir migrações perfeitas de várias fontes de banco de dados para plataformas de dados do Azure com tempo mínimo de inatividade.

| **Recurso** | **Limite** | **Comentários** |
| --- | --- | --- |
| Número máximo de serviços por assinatura, por região |2 | Para solicitar um aumento para este limite, entre em contato com o suporte. |
