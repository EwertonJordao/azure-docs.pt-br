## <a name="query-data"></a>Consultar dados

1. Selecione o botão **v** para abrir a consulta. Faça uma cópia para a interface do usuário da Web para editar a consulta.

1. No menu à direita, selecione **Consultas rápidas** e **Ferramentas**: 

    * A opção **Consultas rápidas** inclui links para a interface do usuário da Web com algumas consultas de exemplo.
    * A opção **Ferramentas** inclui um link para **comandos DROP** na interface do usuário da Web, que permite solucionar problemas executando os comandos `.drop` relevantes.

    > [!TIP]
    > Você poderá perder dados ao usar comandos `.drop`. Use-os com atenção.
