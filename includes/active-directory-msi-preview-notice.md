Identidades gerenciadas para recursos do Azure é um recurso do Azure Active Directory. Cada um dos [serviços do Azure que dão suporte a identidades gerenciadas para recursos do Azure](/azure/active-directory/managed-identities-azure-resources/services-support-msi) está sujeito à própria linha do tempo.
