[O RBAC (Azure Role-based Access Control, controle de acesso baseado em função)](../articles/role-based-access-control/overview.md) é o sistema de autorização que você usa para gerenciar o acesso aos recursos do Azure. Para determinar a quais recursos usuários, grupos, diretores de serviço ou identidades gerenciadas têm acesso, você lista suas atribuições de função.
