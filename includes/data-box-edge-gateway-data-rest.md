Para dados em repouso:

- O acesso aos dados armazenados em compartilhamentos é restrito.

    - Os clientes SMB que acessam dados compartilhados precisam de credenciais de usuário associadas ao compartilhamento. Essas credenciais são definidas quando a ação é criada.
    - Os endereços IP dos clientes NFS que acessam uma ação precisam ser adicionados quando a ação é criada.
