Se você estiver usando o Azure CloudShell, você estará conectado à sua conta automaticamente e não precisará executar o comando Connect-AzAccount. Se você estiver executando o PowerShell localmente, abra o console do PowerShell com privilégios elevados e conecte-se ao Azure usando o comando `Connect-AzAccount`.
