## <a name="delete-anchors"></a>Excluir âncoras

Para excluir uma âncora espacial da nuvem, use o método `DeleteAnchor()`. Excluir âncoras quando elas não forem mais usadas é uma boa prática para incluir no início do seu processo e práticas de desenvolvimento para manter os recursos do Azure limpos.
