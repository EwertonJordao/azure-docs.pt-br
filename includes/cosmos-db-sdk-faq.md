**Como serei notificado sobre o SDK de desativação?**

A Microsoft fornecerá o aviso prévio de 12 meses antes do fim do suporte do SDK de desativação para facilitar uma transição tranqüila para um SDK com suporte. Você será notificado por meio de vários canais de comunicação: o portal do Azure, as atualizações do Azure e a comunicação direta com os administradores de serviço atribuídos.

**Posso criar aplicativos usando um SDK para Azure Cosmos DB a ser desativado durante o período de 12 meses?** 

Sim, você poderá criar, implantar e modificar aplicativos usando o SDK de Azure Cosmos DB a ser desativado durante o período de aviso de 12 meses. Recomendamos que você migre para uma versão mais recente com suporte do SDK do Azure Cosmos DB durante o período de aviso de 12 meses, conforme apropriado. 

**Após a data de desativação, o que acontece com os aplicativos que usam o SDK de Azure Cosmos DB sem suporte?** 

Após a data de desativação, o Azure Cosmos DB não fará mais correções de bugs, adicionará novos recursos ou dará suporte às versões descontinuadas do SDK. Se você preferir não atualizar, as solicitações enviadas das versões desativadas do SDK continuarão sendo servidas pelo serviço de Azure Cosmos DB. 

**Quais versões do SDK terão os recursos e as atualizações mais recentes?**

Novos recursos e atualizações serão adicionados somente à versão secundária mais recente da versão mais recente do SDK com suporte. Recomendamos que você sempre use a versão mais recente para aproveitar os novos recursos, aprimoramentos de desempenho e correções de bugs. Se você estiver usando uma versão antiga e não desativada do SDK, suas solicitações para Azure Cosmos DB continuarão a funcionar, mas você não terá acesso a novos recursos.  

**O que devo fazer se eu não conseguir atualizar meu aplicativo antes de uma data de fechamento?**

Recomendamos atualizar o mais rápido possível para o SDK mais recente. Depois que um SDK estiver marcado para desativação, você terá 12 meses para atualizar seu aplicativo. Se você não conseguir atualizar pela data de aposentadoria, as solicitações enviadas das versões desativadas do SDK continuarão a ser servidas pelo Azure Cosmos DB, para que os aplicativos em execução continuem a funcionar. Mas o Azure Cosmos DB não fará mais correções de bugs, adicionará novos recursos ou dará suporte às versões desativadas do SDK. 

Se você tiver um plano de suporte e precisar de suporte técnico, [Fale conosco](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview) ao arquivar um tíquete de suporte.
