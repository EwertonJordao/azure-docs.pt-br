> [!NOTE]  
> O fuso horário padrão usado para executar expressões CRON é o Tempo Universal Coordenado (UTC). Para que sua expressão CRON seja executada com base em outro fuso horário, crie uma configuração de aplicativo para o seu aplicativo de função chamado WEBSITE_TIME_ZONE. Para saber mais, consulte [os fusos horários NCRONTAB](../articles/azure-functions/functions-bindings-timer.md#ncrontab-time-zones).
