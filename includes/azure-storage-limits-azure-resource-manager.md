Os limites a seguir se aplicam somente quando você executa operações de gerenciamento usando Azure Resource Manager com o armazenamento do Azure.

| Recurso | Limite |
| --- | --- |
| Operações de gerenciamento de conta de armazenamento (leitura) |800 a cada 5 minutos |
| Operações de gerenciamento de conta de armazenamento (gravação) |10 por segundo/1200 por hora |
| Operações de gerenciamento de conta de armazenamento (lista) |100 a cada 5 minutos |
