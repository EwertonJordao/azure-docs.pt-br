1. Faça login no servidor de processo como administrador.
2. Baixe a versão mais recente do Instalador de [Configuração Unificado](https://aka.ms/unifiedinstaller).
3. Clique duas vezes o instalador para iniciar o processo de atualização.
4. O instalador detecta os componentes de recuperação do site instalados e os atualiza para a versão mais recente.
