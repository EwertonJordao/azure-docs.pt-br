## <a name="prepare-your-environment"></a>Prepare o seu ambiente

1. Para executar comandos da CLI do Azure, faça uma [instalação local](/cli/azure/install-azure-cli) da CLI do Azure ou inicie o [Azure Cloud Shell](/azure/cloud-shell/quickstart) usando o ambiente bash.

1. Se estiver usando uma instalação local, entre com a CLI do Azure usando o comando [az login](/cli/azure/reference-index#az-login).  Siga as etapas exibidas em nosso terminal para concluir o processo de autenticação.  Confira [Entrar com a CLI do Azure](/cli/azure/authenticate-azure-cli) para obter outras opções de entrada.
