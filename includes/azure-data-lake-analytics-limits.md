O Azure Data Lake Analytics facilita a complexa tarefa de gerenciar infra-estrutura distribuída e código complexo. Ele dinamicamente provisiona recursos, e você pode usá-lo para fazer análises sobre exabytes de dados. Quando o trabalho é concluído, ele acaba com os recursos automaticamente. Você paga apenas pelo poder de processamento que foi usado. À medida que você aumenta ou diminui o tamanho dos dados armazenados ou a quantidade de computação usada, você não precisa reescrever o código. Para aumentar os limites padrão para sua assinatura, entre em contato com o suporte.

| **Recurso** | **Limite** | **Comentários** |
| --- | --- | --- |
| Número máximo de trabalhos simultâneos |20 | |
| Número máximo de unidades de análise (UA) por conta |250 | Use qualquer combinação até o máximo de 250 AUs em 20 trabalhos. Para aumentar esse limite, entre em contato com o Suporte da Microsoft. |
| Tamanho máximo de script para envio de trabalho | 3 MB | |
| Número máximo de contas do Data Lake Analytics por região por assinatura | 5 | Para aumentar esse limite, entre em contato com o Suporte da Microsoft. |
