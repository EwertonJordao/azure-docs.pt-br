Na página **Azure VPN - Propriedades,** configure configurações de login.

1. Definir ativado para que os usuários **Yes** **entrem?** Esta configuração permite que todos os usuários do inquilino AD se conectem à VPN com sucesso.
2. Definir **a atribuição de usuário necessária?** para **Sim** se você quiser limitar o login apenas aos usuários que têm permissões para o Azure VPN.
3. Salve suas alterações.
