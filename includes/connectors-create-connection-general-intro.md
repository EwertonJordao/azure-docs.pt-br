Quando você adiciona um gatilho ou uma ação que se conecta a um serviço ou sistema pela primeira vez, o designer do aplicativo lógico solicita que você crie uma *conexão* fornecendo as informações necessárias, que varia de acordo com a conexão, por exemplo:

* Um nome a ser usado para a nova conexão
* O nome do servidor ou do sistema
* Suas credenciais de conta ou usuário
* O tipo de autenticação a ser usado
* A assinatura e o nome do Azure para o gateway de dados que você criou anteriormente ao se conectar a fontes de dados locais
