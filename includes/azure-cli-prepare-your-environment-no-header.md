- Preparar o ambiente para a CLI do Azure:

  - Inicie o [Azure Cloud Shell](/azure/cloud-shell/quickstart) usando o ambiente bash ou, se você quiser executar a CLI do Azure localmente, faça uma [instalação local](/cli/azure/install-azure-cli) da CLI do Azure.

  - Se estiver usando uma instalação local, entre com a CLI do Azure usando o comando [az login](/cli/azure/reference-index#az-login).  Siga as etapas exibidas em nosso terminal para concluir o processo de autenticação.  Confira [Entrar com a CLI do Azure](/cli/azure/authenticate-azure-cli) para obter outras opções de entrada.
  - Quando solicitado, instale as extensões da CLI do Azure no primeiro uso.  Saiba mais sobre as extensões em [Usar extensões com a CLI do Azure](/cli/azure/azure-cli-extensions-overview).
