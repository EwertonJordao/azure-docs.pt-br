> [!IMPORTANT]
> Para evitar cobranças quando nenhum trabalho estiver em execução, **defina os nós mínimos como 0**. Essa configuração permite que Azure Machine Learning desaloque os nós quando eles não estiverem em uso. Qualquer valor maior que 0 manterá esse número de nós em execução, mesmo que eles não estejam em uso.
