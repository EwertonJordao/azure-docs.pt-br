### <a name="open-the-mfa-page"></a><a name="mfa"></a>Abra a página do MFA

1. Entre no portal do Azure.
2. Navegue até **o Azure Active Directory -> Todos os usuários**.
3. Selecione **Autenticação multifatorial** para abrir a página de autenticação de vários fatores.

   ![Entrar](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="select-users"></a><a name="users"></a>Selecionar usuários

1. Na página **de autenticação de vários fatores,** selecione os usuários para os quais deseja ativar o MFA.
2. Selecione **Habilitar**.

   ![Selecionar](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)
