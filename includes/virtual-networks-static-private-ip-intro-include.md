Uma máquina virtual (VM) é automaticamente atribuída a um endereço IP privado a partir de um intervalo que você especifica, com base na sub-rede na qual a VM é implantada. A VM mantém o endereço até que a VM seja excluída. O Azure atribui dinamicamente o próximo endereço IP privado disponível a partir da sub-rede em que você cria uma VM. Se você quiser um endereço IP específico da sub-rede atribuída à VM, atribua um endereço IP estático.
