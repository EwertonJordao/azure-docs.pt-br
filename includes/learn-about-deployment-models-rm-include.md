> [!NOTE]
> O Azure tem dois modelos de implantação diferentes que você pode usar para criar e trabalhar com recursos: [Azure Resource Manager e classic](../articles/azure-resource-manager/management/deployment-models.md). Este artigo abrange o uso do modelo de implantação do Gerenciador de Recursos. Recomendamos o modelo de implantação do Resource Manager para novas implantações em vez do modelo clássico de implantação.
