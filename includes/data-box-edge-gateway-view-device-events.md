Execute as etapas a seguir no portal do Azure para exibir um evento de dispositivo. 

1. Na portal do Azure, vá para o recurso de Azure Stack borda/Gateway do Data Box e vá para **monitoramento > eventos de dispositivo**.
2. Selecione um evento e exiba os detalhes do alerta. Tome as devidas providências para resolver a condição de alerta.

    ![Selecione o evento e exiba os detalhes](media/data-box-edge-gateway-view-device-events/view-device-events.png)
