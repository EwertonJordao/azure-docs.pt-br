- Para acessar o serviço Data Box Edge, sua organização precisa ter uma assinatura de Enterprise Agreement (EA) ou Cloud Solution Provider (CSP). Para obter mais informações, consulte [Inscreva-se para uma assinatura do Azure](https://azure.microsoft.com/resources/videos/sign-up-for-microsoft-azure/).
- Como este serviço de gerenciamento está hospedado no Azure, ele é protegido pelos recursos de segurança do Azure. Para obter mais informações sobre os recursos de segurança fornecidos pelo Azure, acesse o [Microsoft Azure Trust Center](https://azure.microsoft.com/support/trust-center/security/).
- Para operações de gerenciamento de SDK, você pode obter a chave de criptografia para o seu recurso em **propriedades do dispositivo**. Você só pode visualizar a chave de criptografia se tiver permissões para a API do Gráfico de Recursos.
